# Конфигурации глушителей

Складывайте PNG-файлы конфигураций в эту папку: `images/configurations/`.

## Файл по умолчанию
Если для выбранного типа нет явного файла, сайт использует:

- `oval-120x265-1in-1out-center.png`

## Текущие имена, используемые в `index.html`

- `round-d170.png`
- `resonator-d100.png`
- `oval-c-c.png`
- `oval-h.png`
- `oval-u.png`
- `oval-l.png`

Вы можете переименовать файлы и обновить объект `productImages` в `index.html` под вашу фактическую номенклатуру.
